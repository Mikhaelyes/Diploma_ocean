Аннотация
В связи с развитием методов нейросетевого прогнозирования океана актуальной задачей становится количественная оценка их эффективности в прикладных сценариях. Настоящее исследование посвящено построению методологии такой оценки на примере реального разлива нефти, произошедшего 29 августа 2025 г. в акватории Новороссийска. На основе спутниковых данных восстановлена динамика пятна, которая используется в качестве эталона. Модельная динамика рассчитывается различными способами: с использованием полей течений из реанализа GLORYS12 и ветрового воздействия ERA5, а также с заменой полей течений краткосрочным прогнозом гидродинамического моделирований и прогнозом при помощи нейронной сети, обученной на исторических данных GLORYS12. Анализ проводится на временном интервале до 72 часов. В результате работы проведено сравнение точности прогноза при использовании различных методов моделирования океанской динамики.
